# Monkeypox data analysis
 Monkeypox viral disease that can be spread between people or between people and certain animals, which has become a serious global pandemic since early 2022. 
This [Monkeypox data](https://github.com/globaldothealth/monkeypox) is gathered by Global Health team in a 100-day mission to provide decision makers, researchers, and the public with timely and accurate, openly-accessible, global line-list data for the 2022 monkeypox outbreak.
